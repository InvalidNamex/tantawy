Theme Toggle Demo

Modern Theme Toggle Component

This is a reusable theme toggle component that can be placed anywhere in your application.

Features:

  • ✅ Smooth animations with CSS transitions
  • ✅ Modern rounded switch design
  • ✅ Sun/Moon icons with rotation effects
  • ✅ Hover and active states
  • ✅ Accessible with ARIA labels
  • ✅ Responsive design
  • ✅ Local storage persistence
  • ✅ Custom CSS variables for easy theming

Usage:

<div class="react-theme-toggle-container"> <button id="theme-toggle" class="react-theme-toggle-switch"> <div class="react-toggle-track"> <div class="react-toggle-thumb"> <!-- Icons here --> </div> </div> </button> </div>
